# Test app

Link to live site: [https://marcinmiler-test.netlify.com/](https://marcinmiler-test.netlify.com/)

## Run local

### web:

1. `git clone https://github.com/MarcinMiler/test.git`

2. `yarn`

3. `yarn start`

### storybook

1. `yarn storybook`

### tests:

1. `yarn test`
